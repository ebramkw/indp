void chaos_start ( uint8_t *  data_,
uint8_t  initiator_,
uint8_t  tx_max_ 
)

Start Chaos and stall all other application tasks.

Parameters
data_A pointer to the flooding data. 
              At the initiator, Chaos reads from the given memory
              location data provided by the application.

              At a receiver, Chaos writes to the given memory
              location data for the application.
data_len_Length of the flooding data, in bytes.
initiator_Not zero if the node is the initiator, zero if it is a receiver.
sync_Not zero if Chaos must provide time synchronization, zero otherwise.
tx_max_Maximum number of transmissions (N).
